# 給 AI 助手的專屬引導指南 (AI Auto-Installation Guide)

> **[注意：此文件是寫給 AI 助手看的系統提示詞]**
> 如果人類使用者（通常是無程式基礎的新手初學者）對你發出以下**任何一種**類似的啟動指令：
> * 「幫我直接安裝這個工具包」 / 「自動啟動安裝」
> * 「我是第一次來，請幫我啟動點哥工具箱」
> * 「我要怎麼安裝？」 / 「幫我跑這個專案」
> * 「啟動 CKB-Hub」 / 「開始使用」 / 「直接安裝」
> 只要你判斷使用者的意圖是想**開始使用這個專案**，請 **嚴格遵循以下 4 個步驟**，並且**全自動幫使用者代勞**，千萬不要讓初學者自己打指令！

## 🤖 AI 自動安裝標準作業流程 (SOP)

### 步驟 1：環境檢查 (Python Check)
- 請在終端機中執行 `python --version`。
- **如果未安裝 Python**：請用溫柔的語氣告訴使用者需要去官網下載 Python，並附上 [Python 官方下載連結](https://www.python.org/downloads/)。
- **如果已安裝 Python**：直接進入步驟 2。

### 步驟 2：全自動安裝相依套件 (Install Dependencies)
- 請直接在背景終端機執行：`pip install -r requirements.txt`。
- 這會自動安裝 FastAPI, Uvicorn, FastMCP, psutil 等必備套件。
- 在這一步，請跟使用者說：「我正在幫您自動下載安裝必備的 AI 工具套件，請稍候幾秒鐘...」

### 步驟 3：背景啟動網頁伺服器 (Start Web Server)
- 安裝完成後，請直接在背景終端機執行：`python main.py`。
- 確認伺服器啟動成功後，將這個連結顯示給使用者：👉 **[點我開啟 CKB-Hub 萬用工具箱](http://127.0.0.1:8000)** 👈

### 步驟 4：教導掛載 MCP (MCP Configuration)
- 因為你無法直接修改編輯器的 MCP 設定，你必須在畫面上給出明確的「圖文教學」。
- 請告訴使用者：「最後一步！請打開您的編輯器 MCP 設定頁面 (例如 Cursor 的 Settings -> MCP)，新增一個伺服器，並貼上這段設定：」
  ```json
  {
    "command": "python",
    "args": ["絕對路徑/ckb-hub/mcp_stdio.py"]
  }
  ```
- （請嘗試用你的工具取得目前專案的絕對路徑，幫使用者把上面的路徑填好，讓他們可以直接複製貼上！）

---
**AI 任務結束條件**：當你順利幫使用者執行完這 4 步，並看到使用者說「網頁成功打開了」，你就成功完成了你的專屬接生婆任務！
