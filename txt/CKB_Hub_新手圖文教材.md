# CKB-Hub 點哥 AI 萬用工具箱：新手完全制霸手冊

歡迎來到 CKB-Hub 的世界！這是一份專為「完全沒有程式背景的初學者」所精心設計的圖文保姆級教材。
在這裡，你會學到如何將你的 AI（如 Cursor 或 Antigravity）升級為擁有 18 項超強專業技能的「神級開發助理」。

## 為什麼我們需要 CKB-Hub？（新手必看）

在學習讓 AI 幫你寫程式時，你是不是曾經遇過以下這些崩潰的狀況：
1. **AI 亂下指令把電腦搞壞**：明明只是想上傳一個簡單的靜態網站，AI 卻擅自打了一堆你看不懂的黑盒子終端機指令（像是安裝一堆怪東西），結果網站沒上去，電腦還跑出一堆紅字錯誤？
2. **每次都要重新教 AI 怎麼做事**：每次要部署 Cloudflare 或是把進度備份到 GitHub，都要去翻以前的舊筆記，把超長一段話複製貼上，重新跟 AI 解釋半天？
3. **網路上專業工具包太難裝**：網路上確實有很多工程師寫的超強外掛系統（例如官方的 MetaMCP 或 Docker 容器），但光是「安裝環境」這一步，就足以讓文科生或初學者直接放棄？

**CKB-Hub 就是為了拯救初學者而誕生的！**
你可以把它想像成是 AI 的「**外掛遙控器**」或「**超強記憶吐司**」。你不需要背誦任何複雜的指令，只要打開我們為你準備的專屬網頁、按下一顆按鈕，你的 AI 就會「瞬間學會」那個領域所有的專業知識與最高安全守則！AI 再也不會亂搞，而是會乖乖照著點哥幫你寫好的完美劇本走。

## 🗺️ 一圖看懂：安裝與運作原理

別擔心安裝很困難，我們把所有的複雜操作都「封裝」起來了。你只需要把資料夾丟給 AI，剩下的它會全自動幫你搞定。

> **👇 這是你的專屬安裝與運作流程圖：**
![安裝流程圖](./installation_flowchart.png)

### 超簡單流程解析：
1. **第一步：無腦安裝**
   把這個 `ckb-hub` 工具包資料夾丟進你的 AI 編輯器（如 Cursor），然後對 AI 說：「請閱讀 `AI_READ.md` 幫我安裝」。AI 就會自動啟動無腦升級腳本。
2. **第二步：呼叫網頁控制台**
   安裝完成後，AI 會幫你在電腦背景開啟一個絕對安全的網頁（網址固定是 `http://127.0.0.1:8000`）。這個網頁架設在你自己電腦上，沒有網路也能開！
3. **第三步：點擊開關，注入靈魂**
   在網頁上，你會看到很多開關（例如「Netlify 一鍵部署」）。只要你點擊打開，系統就會在 0.1 秒內，把點哥寫好的「超級指令與安全守則（Markdown）」悄悄注入 AI 的大腦。
4. **第四步：享受自動化工作**
   現在，你只要對 AI 說「幫我部署到 Netlify」，它立刻就能幫你完美、零錯誤地完成工作！

## 🦸‍♂️ 18 項神級技能大公開（完整解說）

打開你的控制台網頁，你會看到像「App Store」一樣的精美面板。目前我們內建了六大分類，共 18 種強大技能，以下為你詳細解說它們的用途：

### 1️⃣ 基礎設定與助理 (每天工作必開)
* **01. 點哥專案助理 (核心)**：這是你的貼身秘書！只要對 AI 說「**開工**」，它就會幫你整理環境、報告進度；說「**收工**」，它會幫你總結今天做了什麼並準備備份。這是初學者必開的神器。
* **API 金鑰管家**：保護你的隱私與錢包！在寫程式時，我們常會用到密碼或金鑰（例如 OpenAI 金鑰）。打開這個，AI 會確保你的機密資訊被安全地鎖在 `.env` 檔案裡，絕對不會被不小心上傳到公開網路上。

### 2️⃣ 靜態與主機部署 (讓全世界看見你的作品)
* **Cloudflare 網頁部署**：高速發布神器！想做一個臨時網址給朋友看你的心血結晶？AI 會幫你使用 Cloudflare Pages 瞬間發布。
* **Netlify 一鍵部署**：最受歡迎的靜態網站發布通道。
* **GitHub 自動備份**：AI 常常打錯 Git 指令導致版本大亂？打開這個開關，AI 就會呼叫最穩定的備份程式，從此告別 Git 紅字報錯。
* **Google Apps Script 部署**：讓 AI 幫你把自動化腳本寫進 Google 雲端。
* **自訂伺服器部署 (SSH/RSYNC)**：如果你自己買了一台 Linux 主機，AI 會變成專業網管，教你安全地上傳檔案。
* **傳統虛擬主機 FTP / PHP 專案部署**：提供給使用傳統虛擬空間（如 CPanel）的同學，自動幫你打包上傳，還會聰明地避開不需要上傳的巨大暫存資料夾。

### 3️⃣ 雲端資料庫 (讓網站擁有記憶)
* **Supabase 資料庫**：想做一個能登入、能留言的網站？打開開關，AI 就會變成 PostgreSQL 資料庫專家，帶你一步步串接目前最火紅的開源資料庫。
* **Firebase 服務**：Google 的王牌資料庫與驗證服務，協助你初始化 SDK 確保連線順暢。

### 4️⃣ AI 工具與模型 (魔法大擴充)
* **匯出至 NotebookLM**：想讓 Google 超強的 NotebookLM 幫你分析專案？AI 會把整個專案的程式碼打包成一份像書本一樣容易閱讀的文字檔。
* **免費 AI API 串接**：教你如何免費申請 API Key，並在你的網頁中快速放入 Google Gemini 或極速版 Groq (Llama 3) AI 機器人，同時教你怎麼寫測試程式！

### 5️⃣ 專案與知識庫 (打造你的第二大腦)
* **Obsidian 知識庫同步**：自動把每天的開發歷程與除錯經驗，轉化為 Obsidian 雙向連結筆記，讓你累積知識不費力。
* **專案知識庫導航**：為整個專案寫好「導讀地圖」。未來就算你換了一台電腦、或是過了一個月才回來寫程式，AI 也能在一秒鐘內看懂整個專案的架構。

### 6️⃣ 診斷與維護 (遇到 Bug 就找他們)
* **專案健檢醫生**：覺得程式卡卡的？幫你的專案做全身健康檢查，找出效能瓶頸。
* **套件衝突排解專家**：遇到 npm 或 pip 安裝失敗、紅字滿天飛的套件地獄時，開啟它！
* **疑難排解大師**：卡關超過 30 分鐘解不出來？開啟此模式，AI 會進入深度除錯狀態，冷靜分析錯誤日誌，並給出超過 3 種以上的突破方案。

## 🧩 終極大絕招：全自動外掛尋找模式 (擴充新技能)

**「如果點哥寫的這 18 個技能還不夠用，我想要自己發明新技能怎麼辦？」**

別擔心！CKB-Hub 擁有最強大的「**全自動模組化外掛系統**」。你完全不用懂任何 Python 或 HTML 程式碼，連改網頁都不用！

假設你想加掛一個名為「一鍵發文到 IG」的新功能，請跟著做：

1. **尋找樣板**：打開工具包裡的 `skills/` 資料夾，你會看到一個 `SKILL_TEMPLATE.md` 檔案，複製它並改名為 `ig_post.md`。
2. **填寫身分證**：打開 `ig_post.md`，在檔案最上方的區塊（這叫做 YAML 身分證）填寫你要顯示在網頁上的資訊：
   ```yaml
   ---
   id: ig_post
   title: 一鍵發文到 IG
   description: 幫我把寫好的文章自動排版並發布到 Instagram。
   category: 基礎設定與助理
   ---
   ```
3. **撰寫神級指令**：在身分證下方，用大白話寫下你希望 AI 遵守的規則與對話劇本（例如：「請先幫我加上 Hashtag，再詢問我是否要發布...」）。
4. **見證奇蹟**：存檔後，**重新整理你的 CKB-Hub 網頁**。你會發現網頁的「基礎設定與助理」分類下，已經**全自動長出了這顆新按鈕**！只要把它打開，AI 就學會這招了！

> **✨ 小撇步**：未來你甚至可以直接對 AI 說：「請參考 `skills/SKILL_TEMPLATE.md` 的格式，幫我寫一個名為『小紅書產生器』的新技能並存檔」，AI 就會自己幫你寫好外掛！
