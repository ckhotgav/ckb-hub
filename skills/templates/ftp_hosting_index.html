概念小教室

🌐 網站主機
就像在網路上「租一塊地來蓋房子」——主機就是那塊地，你的網頁檔案和資料庫都放在裡面。
📁 FTP
就像網路的「檔案總管」，把你的電腦檔案傳送到主機上。傳上去之後，全世界都能看到你的網站。
⚡ PHP
讓網頁「會動」的程式語言，可以從資料庫撈資料即時顯示。
🗄️ MySQL
存放網站資料的軟體（會員名單、文章內容等）。PHP 負責讀寫，MySQL 負責儲存。
